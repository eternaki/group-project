# Deliverables - Dokumentacja Projektu

Ten folder zawiera dokumenty do skopiowania do szablonów Word w `template/`.

## Struktura

| Plik źródłowy (MD) | Szablon docelowy (DOC) |
|--------------------|------------------------|
| `DPP.md` | `PG_WETI_DPP_wer. 1.00.doc` |
| `Specyfikacja.md` | `PG_WETI_DTP_wer. 1.00.doc` |
| `Plakat.md` | `PG_WETI_Plakat.doc` |
| `Raport-roczny.md` | (szablon do pobrania) |
| `Prezentacja.md` | (PowerPoint) |

## Workflow

1. Agent AI edytuje pliki `.md`
2. Człowiek kopiuje treść do szablonu Word
3. Formatowanie według wytycznych WETI
