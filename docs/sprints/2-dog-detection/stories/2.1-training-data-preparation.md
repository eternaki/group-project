## YOLO Dataset Structure

```
data/bbox_training/
├── images/
│   ├── train/
│   ├── val/
│   └── test/
├── labels/
│   ├── train/
│   ├── val/
│   └── test/
└── dataset.yaml
```

**dataset.yaml:**
```yaml
path: data/bbox_training
train: images/train
val: images/val
test: images/test

names:
  0: dog
```

**Label format (per image):**
```
<class_id> <x_center> <y_center> <width> <height>
0 0.5 0.5 0.3 0.4
```
