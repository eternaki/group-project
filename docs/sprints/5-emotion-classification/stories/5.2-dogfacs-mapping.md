## DogFACS Action Units (Relevant)

| AU | Name | Description |
|----|------|-------------|
| AU101 | Inner Brow Raiser | Raises inner portion of brow |
| AU102 | Outer Brow Raiser | Raises outer portion of brow |
| AU201 | Ear Flattener | Flattens ears against head |
| AU301 | Nose Wrinkler | Wrinkles nose bridge |
| AU401 | Upper Lip Raiser | Exposes upper teeth |

---

## Emotion to AU Mapping

| Emotion | Primary AUs | Description |
|---------|-------------|-------------|
| happy | Relaxed ears, open mouth | No tension AUs active |
| sad | AU201, lowered brow | Flattened ears, droopy expression |
| angry | AU301, AU401, tense ears | Wrinkled nose, bared teeth |
| fearful | AU201, wide eyes | Flattened ears, tense body |
| relaxed | Soft features | Minimal AU activation |
| neutral | Baseline | Default resting state |
