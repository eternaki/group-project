# Sprint 8: Data Collection

**Sprint Goal:** Collect 2,500 YouTube videos of dogs showing emotions.

**Duration:** Weeks 16-24
**Semester:** 2
**Phase:** Dataset Creation

---

## Overview

This sprint focuses on collecting YouTube videos of dogs displaying various emotions. Videos will be downloaded, preprocessed, and organized for annotation.
